# almoxarifado-3b
Sistema de almoxarifado para o SENAI feito pela turma 3B

## Objetivos do projeto:
Este sistema foi desenvolvido na discplina de Implantação de Sistemas do SENAI. O objetivo é controlar o estoque de materiais de TI e
do escritório da instituição, registrando entradas, saídas e gerando alertas de estoques baixos.
