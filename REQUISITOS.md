# Requisitos de Sistema de Almoxarifado
## Requisitos Funcionais
* RF01: Cadastro do usuário
* RF02: Cadastro do produto
* RF03: Registro da entrada
* RF04: Registro da saída
* RF05: Alerta de estoque insuficiente

## Requisitos Não Funcionais
* RNF01: Responsividade do sistema
* RNF02: Desempenho do sistema
* RNF03: Interface amigável
